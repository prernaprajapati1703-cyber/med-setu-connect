🩺 MedSetu

Bridging Healthcare Through Technology

MedSetu is a healthcare platform designed to connect patients with doctors and essential healthcare services. The goal is to make healthcare more accessible, organized, and user-friendly through digital innovation.

🚀 Features

- 👤 Secure user authentication
- 🩺 Doctor and patient profiles
- 📅 Appointment booking
- 💊 Medicine reminders
- 🌍 Multilingual support
- 📍 Nearby hospitals and pharmacies
- 📄 Digital medical records
- 🚑 Emergency assistance
- 🔔 Health notifications
- 🤖 AI-powered health guidance

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- React
- Firebase
- AI Integration (Planned)

📂 Project Status

🚧 This project is currently under development. New features and improvements will be added regularly.

🎯 Vision

To bridge the gap between patients and healthcare providers by offering a simple, secure, and smart digital healthcare platform.
